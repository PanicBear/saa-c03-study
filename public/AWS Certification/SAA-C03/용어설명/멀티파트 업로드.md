→ 멀티파트 업로드를 사용하면 단일 객체를 여러 부분의 집합으로 업로드할 수 있다.

각 부분은 객체 데이터의 연속적인 부분이며, 객체 부분은 독립적으로 그리고 임의의 순서로 업로드할 수 있다.

부분의 전송이 실패할 경우 다른 부분에 영향을 주지 않고도 해당 부분을 재전송할 수 있다. 객체의 모든 부분이 업로드되면 Amazon S3가 이들 부분을 수집하여 객체를 생성한다.
